Sobre esta aplicação

1. Arquitetura MVT do Django

Esta aplicação foi desenvolvida usando a arquitetura MVT do Django:

  • Model → representa os dados da aplicação.
  • View → trata a lógica da aplicação.
  • Template → apresenta as páginas HTML.

O utilizador faz pedidos através do navegador. As views recebem esses pedidos, comunicam com os models e devolvem páginas HTML usando templates.

2. Modelação

A aplicação possui modelos para:

  • Projetos
  • Tecnologias
  • Competências
  • Formações

Os modelos estão definidos no ficheiro models.py.
